# SHAKTI Frontend

This package contains the SHAKTI SAHAYATA web interface built with React, TypeScript, Vite, shadcn/ui, and Aceternity-inspired components.

## Main Project Guide

Use the root repository README for full setup instructions:

- [../README.md](../README.md)

## Frontend-Only Commands

From the repository root:

```powershell
npm run dev:frontend
```

```powershell
npm run build --workspace=frontend
```

```powershell
npm run test --workspace=frontend
```
